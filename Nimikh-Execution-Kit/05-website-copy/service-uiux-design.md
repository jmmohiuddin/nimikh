## Section: FAQ

**How do your design systems integrate with developer codebases?**
We export Figma design tokens to a centralized JSON file that compiles directly into Tailwind CSS variables via automated Git scripts. When a designer updates a color in Figma, a PR opens in engineering with the diff — reviewed, tested by Chromatic, merged, deployed.

**Can you retrofit a design system onto our existing codebase?**
Yes. We audit your current UI, extract implicit tokens, propose the migration plan, and roll out per-component without a full rebuild.

**What if we don't use Tailwind?**
The token pipeline outputs CSS custom properties, SCSS variables, or JS design-token files — Tailwind is a preference, not a requirement.

**How do you measure design system adoption?**
Storybook coverage, deprecated-component usage in production (via ESLint rules), and design-review latency before/after.
